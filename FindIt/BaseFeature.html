    class FindIt::BaseFeature

    Abstract class for a map feature.

    To implement a feature, the derived class must:

    • Either define @type class instance variable or override self.type method.

    • Either define @marker class instance variable or override self.marker method.

    • Either define class instance variable or override self.marker_shadow method.

    • Override self.closest method.

    Example:

    class Library < FindIt::BaseFeature
  @type = :LIBRARY
  @marker = FindIt::MapMarker.new(
    "http://maps.google.com/mapfiles/kml/pal3/icon56.png",
    :height => 32, :width => 32).freeze
  @marker_shadow = FindIt::MapMarker.new(
    "http://maps.google.com/mapfiles/kml/pal3/icon56s.png",
    :height => 32, :width => 59).freeze
  def self.closest(loc)
       . 
       . 
       . 
  end
end

    Attributes

    address[R]

    The street address of this feature. This value is required and will always be defined.

    city[R]

    The city portion of the address of this feature. This value is required and will always be defined.

    distance[R]

    A Float value containing the distance (in miles) from the origin point to this feature.

    link[RW]

    A URL to associate with this feature, such as a home page. This value is optional and may be nil.

    location[R]

    A FindIt::Location instance that provides the geographical location (latitude, longitude) of this feature. This value is required and will always be defined.

    name[R]

    The name of this feature, such as “Grouch Marx Middle School”. This value is optional and may be nil.

    note[RW]

    An optional note with additional information about this feature. This value is optional and may be nil.

    state[R]

    The state portion of the address of this feature. This value is required and will always be defined.

    title[R]

    A title for this feature, such as “Closest library”. This value is required and will always be defined.

    zip[R]

    The postal code portion of the address of this feature. This value is optional and may be nil.

    new(location, params = {}) click to toggle source

    Construct a new feature.

    Parameters:

    • location – A FindIt::Location instance with the location of this feature.

    • params – Parameters to initialize feature attributes. See the description of the FindIt::BaseFeature attributes for details on parameter values.

    Returns: A FindIt::BaseFeature instance.

    The following parameters are required:

    • :title => String

    • :address => String

    • :city => String

    • :state => String

    One of the following parameters must be specified.

    • :distance => Numeric

    • :origin => FindIt::Location

    If both are specified, the :distance takes precedence. The :origin is used only to calculate distance; the coordinate itself is not saved.

    The following parameters are optional.

    • :name => String

    • :zip => String

    • :link => String

    • :note => String

    # File lib/findit/base-feature.rb, line 109
def initialize(location, params = {})
  [:title, :address, :city, :state].each do |p|
    raise "required parameter \":#{p}\" not specified" unless params.has_key?(p)
  end
  @location = location
  @title = params[:title]
  @name = params[:name] if params.has_key?(:name)
  @address = params[:address]
  @city = params[:city]
  @state = params[:state]
  @zip = params[:zip] if params.has_key?(:zip)
  @link = params[:link] if params.has_key?(:link)
  @note = params[:note] if params.has_key?(:note)
  @distance = if params.has_key?(:distance)
    params[:distance]
  elsif params.has_key?(:origin)
    location.distance(params[:origin])
  else
    raise "must specify either \":distance\" or \":origin\" parameter"
  end
end

    # File lib/findit/base-feature.rb, line 247
def info
  result = []    
  result << "<b>" + @title.capitalize_words.html_safe + "</b>"
  result << @name.html_safe unless @name.empty?
  result << @address.html_safe
  
  s = @city + ", " + @state
  s += " " + @zip unless @zip.empty?
  result << s.html_safe
  
  result += @note.html_safe.split("\n") unless @note.empty?
  
  result << "%.1f mi away" % [@distance]
    
  unless @link.empty?
    result << "<a href=\"" + @link.html_safe + "\">more info ...</a>"
  end
  
  "<div class=\"findit-feature-info\">\n" + result.join("<br />\n") + "\n</div>"
end
